How to fix branding on this box

1. Update the property name. SSH to the staging server and run:

cd /var/www/sheeza-manzil
sudo -u www-data ./venv/bin/flask --app run.py brand show
sudo -u www-data ./venv/bin/flask --app run.py brand set \
    --name "Maakanaa Village Hotel" --short "Maakanaa"
sudo systemctl restart sheeza.service

2. Or use the admin UI: open /admin/property-settings/ and edit Property name + Short name. No restart needed.

3. Verify: reload this page — property_name above should show the new value, and the sidebar / top header will pick it up automatically.
